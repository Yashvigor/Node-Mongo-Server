mongod: core engine of mongodb
mongosh: for connecting mongod but it is command line shell. fetch data from mongod in the form of json.
mongodb compass: gui for connecting mongod, same as mongosh 
- by default runs at 27017 port

- a database can have multiple collections and a collection can have multiple documents which have json like data. 


Mongodb Shell:

1) use databasename; : for switching to a db
2) show dbs; : for showing available databases
3) show collections : for showing available collections
4) db.mycollection.insertOne({key : value, key : value}) : for inserting document in the collection (automatically gives object an id)
5) cls : clearing command shell
6) db.mycollection.insertMany([{key: value,key:value},{key:value,key: value}]) : inserting multiple documents at once in db
